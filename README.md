## Local Deploy

首先拉取代码，然后进入项目目录。

```bash
git clone https://github.com/quant-wiki/quant-wiki.git
cd quant-wiki
```

推荐使用 `venv` 创建虚拟环境，并安装依赖。

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

本地预览只需要运行如下命令即可：

```bash
mkdocs serve
```
