# 量化交易进阶书籍推荐

## 《算法交易和量化策略》
<img src="../images/algo-trading.jpg" alt="算法交易和量化策略" width="200"/>

- **作者**: Ernest P. Chan
- **出版社**: Wiley
- **难度**: ⭐⭐⭐⭐
- **推荐指数**: ⭐⭐⭐⭐⭐

### 内容简介
本书深入探讨了算法交易的核心概念和高级策略，包括统计套利、机器学习应用等前沿主题。

### 核心章节
1. 统计套利理论
2. 配对交易策略
3. 机器学习应用
4. 高频交易技术
5. 风险管理框架

### 实践价值
- 完整的策略开发流程
- 真实案例分析
- Python实现代码
